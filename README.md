# Virtual Pet 🐾
Proyecto React desarrollado con Vite para la gestión de productos de mascotas.

## Historias de Usuario

**HU1 - Visualizar productos**  
Como usuario quiero ver los productos disponibles para conocer lo que ofrece la tienda.  

**HU2 - Navegación entre páginas**  
Como usuario quiero navegar entre inicio, productos y carrito para moverme fácilmente en la aplicación.  

**HU3 - Página de inicio atractiva**  
Como usuario quiero ver una página de inicio clara y visual para entender rápidamente la aplicación.  

**HU4 - Carrito de compras**  
Como usuario quiero agregar y ver productos en el carrito para gestionar mis compras.  

**HU5 - Registro e inicio de sesión**  
Como usuario quiero registrarme e iniciar sesión para acceder a funcionalidades privadas.  

Todas las historias de usuario fueron implementadas y pueden ser verificadas en la aplicación.
